To run the Thirdlight browsing app...

```
npm install
```

Set environment variables:

```
export THIRDLIGHT_URL=<thirdlight_url>
export API_KEY=<api_key>
export USERNAME=<username>
```

Start the app:

```
npm start
```

### Examples
